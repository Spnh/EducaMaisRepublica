Linguagem culta e coloquial



Linguagem culta

A linguagem Culta é aquela que segue as regras estabelecidas por uma língua, o que significa que, segue as regras gramaticais e o acordo ortográfico. Ela também retifica o bom senso de evitar certas expressões. Um exemplo de onde essa linguagem é utilizada é uma conversa de negocio, uma entrevista, etc.


Linguagem Coloquial

Já a linguagem coloquial tem haver com a fala e por isso ela representa uma liberdade de uso. Pois, quando conversamos , tudo é mais veloz, não há tempo para corrigir a fala, pois os pensamentos apenas fluem. Um exemplo de onde essa linguagem é utilizada é conversa com os amigos, uma conversar no bar, etc.


Diferença

A principal diferença entre elas é que a linguagem coloquial é uma linhagem mais relaxada, não se utiliza de muitas regras de gramatica, sendo muito presente gírias e expressões populares. Já a coloquial se utiliza de muitas regras estritas de gramática, pois são muito utilizadas em documentos oficiais, reuniões e discursos.
